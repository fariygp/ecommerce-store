# 🛍️ MyStore - Frontend E-Commerce Project

A complete responsive e-commerce frontend built with HTML, CSS, and vanilla JavaScript.

## Features
- Product listing page
- Add/remove items from cart
- Persistent cart using LocalStorage
- Responsive design
- Cart total calculation

## How to use
1. Clone the repo
2. Open `index.html` in your browser
3. Browse products and add to cart

## Technologies
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript ES6
- LocalStorage API
